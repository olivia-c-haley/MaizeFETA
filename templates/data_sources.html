Sources

Annotation databases, tools, and genome assemblies underlying FETA's enrichment analyses.

All annotations were generated against specific genome assembly versions and are static snapshots — they are not updated automatically. Where a tool or database has been updated since annotation, results may differ from a live query against that resource. Tags indicate whether a source is available for B73 only or both B73 and NAM lines.

Gene Ontology (GO) Terms

GOMAP B73 only

Gene Ontology Meta Annotator for Plants. Integrates multiple evidence types including sequence homology, domain-based inference, and literature mining to produce high-coverage GO annotations for plant genomes.

Source:Cyverse repo

Version: October_2022_v2.r1

Wimalanathan & Lawrence-Dill, Plant Cell Physiol. 2021 · DOI

EnTAP GO B73 + NAM

GO terms assigned via the Eukaryotic Non-Model Transcriptome Annotation Pipeline (EnTAP), which uses sequence similarity searches against reference protein databases to transfer GO annotations.

Hart et al., Mol. Ecol. Resour. 2020 · DOI

PANNZER GO B73 + NAM

GO terms assigned via the Protein ANNotation with Z-score EnRichment (PANNZER) classifier, which uses weighted K-nearest neighbor approaches to infer functional annotations based on sequence similarity.

Törönen et al., Nucleic Acids Res. 2018 · DOI

GO Expanded B73 + NAM

A non-redundant union of all available GO annotations for each genotype, with full propagation up the GO hierarchy (true path rule). Provides the broadest coverage and is the recommended default for most analyses.

Pathways

EnTAP KEGG B73 only

Kyoto Encyclopedia of Genes and Genomes (KEGG) pathway annotations assigned via EnTAP sequence similarity search against the KEGG protein database.

kegg.jp · Kanehisa et al., Nucleic Acids Res. 2023

CornCyc (MaizeCyc) B73 only

Maize metabolic pathway database derived from the PlantCyc framework. Gene-to-pathway associations curated from literature and computational predictions for maize metabolism.

MaizeGDB CornCyc

Reactome B73 only

Plant pathway annotations from the Reactome knowledgebase, transferred to maize B73 gene models via orthology-based projection.

Milacic et al., Nucleic Acids Res. 2024 · reactome.org

Enzyme Commission (EC)

PANNZER EC B73 + NAM

EC numbers assigned by PANNZER via homology-based annotation against reference enzyme databases.

Törönen et al., Nucleic Acids Res. 2018 · DOI

E2P2 EC B73 only

EC number annotations assigned by the Ensemble Enzyme Prediction Pipeline (E2P2), which uses sequence homology to infer metabolic-related functions.

Schläpfer et al., Plant Physiol. 2017 · DOI

EC Expanded B73 only

Non-redundant union of PANNZER EC and E2P2 EC annotations, with hierarchical propagation to parent EC classes.

Protein Domains

Pfam B73 + NAM

Protein family (Pfam) domain annotations from the Pfam database, assigned by HMMER profile searches against the Pfam-A HMM library.

Mistry et al., Nucleic Acids Res. 2021 · pfam.xfam.org

InterProScan B73 + NAM

Protein domain classifications from the InterProScan software package. It identifies protein families, domains, and functional sites by combining multiple diagnostic resources (e.g., Pfam, PROSITE, SMART,PRINTS, PANTHER, Gene3D) in one workflow.

Paysan-Lafosse et al., Nucleic Acids Res. 2023 · ebi.ac.uk/interpro

Cell Type (scATAC)

scATAC-seq Cell Type Annotations B73 only

Cell type accessibility annotations derived from single-cell ATAC-seq data in maize. Genes are linked to accessible chromatin regions in specific cell types, enabling enrichment of genes preferentially active in particular tissues or cell populations.

add citation

Loci

Loci Annotations B73 only

Named loci, QTL intervals, curated genetic regions, or chromosomal features.

Agronomic Traits (GWAS)

Wallace et al. 2014 GWAS B73 only

Genome-wide association study linking maize genes to agronomic traits. Gene-to-trait associations derived from significant GWAS hits mapped to B73 v5 gene models.

Wallace et al., PLOS Genetics 2014 · DOI

GWAS Atlas Database B73 only

A curated compendium of published maize GWAS results aggregated from the GWAS Atlas resource. Gene-to-trait links are derived from significant associations mapped to B73 v5 gene coordinates.

Canela-Xandri et al., Nature Genetics 2018 · atlas.ctglab.nl

Subcellular Localization

DeepLoc 2.1 B73 + NAM

Deep learning-based subcellular localization predictor. Protein sequences are classified by cellular compartments (e.g.,plastid, mitochondrion, nucleus), signal protein, and solubility.

Ødum et al., Nucleic Acids Res. 2024 DOI

Other Analytics

Phylostratr B73 + NAM

Phylostratigraphic age assignments for maize genes. Each gene is classified into one of 14 phylostrata representing the taxonomic level at which the gene family first appeared, from cellular organisms (stratum 1) to species-specific genes (stratum 14).

Tibbs-Cortez et al., Manuscript forthcoming.

MaizeGDB Phylostratr Tool

PSAURON B73 + NAM

Protein Sequence Assessment Using a Reference ORF Network (PSAURON). Scores for each gene on a 0.0–1.0 scale reflects the likelihood that the sequence is a genuine protein-coding region.

Sommer et al.,NAR Genom. Bioinform. 2025 DOI

RNA-Seq Datasets

Hayford et al. 2024 — MaizeGDB Curated RNA-Seq B73 only

A curated compendium of maize RNA-Seq differential expression datasets aggregated from published studies and deposited in MaizeGDB. Datasets were uniformly processed and mapped to B73 v5 gene models, enabling cross-study comparison of DEGs across diverse tissues, developmental stages, and stress conditions. These datasets underpin the MaizeGDB curated study browser in the RNA-Seq Analysis tool.

Hayford et al., BMC Genomics 2024 · DOI
